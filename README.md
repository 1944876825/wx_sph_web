# 微信视频号 打招呼自动回复 前端

基于 React + TypeScript + Ant Design 的微信视频号自动回复管理后台。

## ✨ 功能特性

- 🔐 微信扫码登录
- 👥 多账号管理
- 📝 自动回复消息配置（文字 + 图片）
- 🤖 **AI 智能回复** — 接入 GLM 大模型，根据用户消息自动生成回复
- ⚙️ AI 配置管理（API Key、模型、系统提示词）
- 📊 账号状态监控

## 技术栈

- React 18 + TypeScript
- Vite
- Ant Design 5
- React Router 6

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

构建产物在 `dist/` 目录，可部署到后端 `public/dist/` 中。
